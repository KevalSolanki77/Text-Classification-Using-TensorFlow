# Text-Classification-Using-TensorFlow
CNN-based sentiment analysis and text classification model using TensorFlow and the IMDB movie review dataset.

## Features

- Text sentiment classification
- IMDB movie review dataset
- Deep learning-based NLP model
- Embedding layer implementation
- Accuracy and loss visualization

## Technologies Used

- Python
- TensorFlow
- Keras
- Matplotlib

## Dataset

The project uses the IMDB movie reviews dataset provided by TensorFlow/Keras.

## Model Architecture

- Embedding Layer
- Conv1D Layer
- GlobalMaxPooling1D
- Dense Layers
- Sigmoid Output Layer
